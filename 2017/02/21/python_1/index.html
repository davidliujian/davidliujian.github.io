a, b, c = 1, 2, "john"

1. 数据类型和变量

数据类型
在Python中，能够直接处理的数据类型有以下几种：
整数、浮点数、字符串布尔值、空值

Python支持四种不同的数字类型
  • int（有符号整型）
  • long（长整型[也可以代表八进制和十六进制]）a=1271837L
  • float（浮点型）
  • complex（复数）complex(a,b),复数的实部a和虚部b都是浮点型
关于字符串转义

转义字符\可以转义很多字符，比如\n表示换行，\t表示制表符，字符\本身也要转义，所以\表示的字符就是\。
Python还允许用r’’表示’’内部的字符串默认不转义
如果字符串内部有很多换行，用\n写在一行里不好阅读，为了简化，Python允许用’’’…’’’的格式表示多行内容

截取字符串 [头下标:尾下标]
+是字符串连接运算符，星号*是重复操作
布尔值可以用and、or和not运算。
空值用None表示。

数据类型转换

  • int(x[,base])
  • long(x[,base])
  • float(x)
  • complex(real)
  • str(x)将对象x转换成字符串
  • repr(x)将对象x转换成表达式字符串

  • eval(x)用来计算在字符串中的有效Python表达式,并返回一个对象

  • tuple(s)将序列 s 转换为一个元组

  • list(s)将序列 s 转换为一个列表
  • set(s)转换为可变集合

  • dict(d)创建一个字典。d 必须是一个序列 (key,value)元组。

  • frozenset(s)转换为不可变集合

  • chr(x) 将一个整数转换为一个字符
  • unichr(x) 将一个整数转换为Unicode字符
  • ord(x) 将一个字符转换为它的整数值
  • hex(x) 将一个整数转换为一个十六进制字符串
  • oct(x) 将一个整数转换为一个八进制字符串

变量

变量名必须是大小写英文、数字和_的组合，且不能用数字开头
可以把任意数据类型赋值给变量，同一个变量可以反复赋值，而且可以是不同类型的变量,这种变量本身类型不固定的语言称之为动态语言

常量

在Python中，通常用全部大写的变量名表示常量,,,只是习惯，可以改变数值

无论整数做除法还是取余数，结果永远是整数

2.list和tuple

list 列表

Python内置的一种数据类型是列表：list。list是一种有序的集合，可以随时添加和删除其中的元素。

classmates = ['Michael', 'Bob', 'Tracy']
len(classmates)

用索引来访问list中每一个位置的元素，记得索引是从0开始的

往list中追加元素到末尾:

classmates.append(‘Adam’)

把元素插入到指定的位置:

classmates.insert(1, ‘Jack’)

删除list末尾的元素:

classmates.pop()

删除指定位置的元素:

classmates.pop(i)

把某个元素替换成别的元素,可以直接赋值给对应的索引位置.
list里面的元素的数据类型也可以不同
L = ['Apple', 123, True]
list元素也可以是另一个list
s = ['python', 'java', ['asp', 'php'], 'scheme']
s[2][0]

tuple 元组

是另一种有序的列表，叫元组。tuple一旦初始化就不能修改

当你定义一个tuple时，在定义的时候，tuple的元素就必须被确定下来
classmates = ('Michael', 'Bob', 'Tracy')

创建一个元素的tuple时，需要加个逗号，不然会被认为时小括号哦~
t = (1,)

t = ('a', 'b', ['A', 'B'])

上面这个tuple中含有列表，列表的内容时可变的，只要tuple还是指向这个列表

4.dict和set

dict，字典，像其他语言的map，使用键-值（key-value）存储，查询快，用空间来换取时间。注意：dict内部存放的顺序和key放入的顺序是没有关系的

d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}
d['Bob'] //75

存入数据
d['jack']=100

一个key只能对应一个value，所以，多次对一个key放入value，后面的值会把前面的值冲掉

通过in判断key是否存在

get方法，若key不存在，可以返回None，或者自己指定的value

pop 要删除一个key，用pop(key)方法，对应的value也会从dict中删除

dict的key必须是不可变对象。
在Python中，字符串、整数等都是不可变的，因此，可以放心地作为key。而list是可变的，就不能作为key

set，无序和无重复元素的集合，只存储key，并且key不能重复

要创建一个set，需要提供一个list作为输入集合

重复元素在set中自动被过滤

add(key)添加元素

remove(key)删除元素

交集&、并集|

set和dict的唯一区别仅在于没有存储对应的value，但是，set的原理和dict一样，所以，同样不可以放入可变对象，因为无法判断两个可变对象是否相等，也就无法保证set内部“不会有重复元素”。
